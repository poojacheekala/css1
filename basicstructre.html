CSS selectors:

A Css selectors selects the Html elements you want to style. selectors are of five types

simple selectors:
  1. element selector
  2. id selector
  3. class selector
  4. universal selector
  5. grouping selector
  1. Element selector:
    2. The element selector selects HTML elements based on the element name.
  2. Id selectors:
    3. Id selector is used to select one unique element. It is represented by"#".
  3. class selectors:
    4. class selectors can have multiple values with space seperated. It is represented bt ".".
  4. Universal Selectors
    5. The universal selectors selects all HTML elements on the page.
  5. Grpouping selectors
    6. Grouping selectors are used to select multiple elements and style them together.
